# Fastabiqoul Khayraat

A React + Firebase Islamic app to promote various religious practices and teachings with real-time database integration.

## Project Structure

- `src/`
  - `components/`
    - `Home.js`
    - `Tilaawat.js`
    - `DalaailulKhayraat.js`
    - `DailyDarood.js`
    - `QuranAudio.js`
    - `Anaasheed.js`
    - `Tafsir.js`
    - `Hadith.js`
    - `Aqeedah.js`
    - `Madhhab.js`
    - `Tariqah.js`
    - `Dhikr.js`
    - `SwalaatTime.js`
    - `Qiblah.js`
    - `HijriCalendar.js`
  - `App.js`
  - `firebaseConfig.js`
- `package.json`

## Features
- Home Page with 3 counters
- 14 Feature Pages
- Real-time Database
- Admin Controls
- User Authentication

## Firebase Configuration
1. Create a Firebase project and add your web app
2. Initialize Firebase in `firebaseConfig.js`
3. Set up Firestore Database for data storage
4. Enable authentication methods as needed.

## Getting Started
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the development server
